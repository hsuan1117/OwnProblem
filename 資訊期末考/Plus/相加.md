# 相加
## Description
Win10是一個每科都很厲害的美女，尤其是數學，更是她的強項。
相加的方法，她一定知道，聰明的你們，能不能追上她呢?

## Input  Format
每筆測資輸入兩個以空格分隔的正整數 ` a 和 b `
	
## Output Format
每筆測資輸出 ` a + b ` (行尾換行)

## Sample Input
  - **Sample Input #1**
  > 1 2  
  
  - **Sample Input #2**
  > 1024 5048  
  
## Sample Output
  - **Sample Output #1**
  > 3
  
  - **Sample Output #2**
  > 6072
